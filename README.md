# BlogMongoDB
Este blog se encuentra disponible para uso en: http://blog-gmatias89.herokuapp.com/
Fue realizado utilizando Node.js Express.js, EJS, HTML5 y CSS3 con MongoDB Atlas como servidor de la base de datos y Heroku 
para levantar la aplicacion, 

Funcionamiento:
Para ingresar un post se le adiciona "/compose" a la ruta de la pagina web por ej: http://blog-gmatias89.herokuapp.com/compose
estos posts se envian a la base de datos y luego se redirige al homepage
